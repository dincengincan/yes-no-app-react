## Run Project Locally

Assumes local installation of [Node.js](https://nodejs.org)

To run the project locally:

- Clone this repository `git clone https://github.com/dincengincan/yes-no-app-react.git`
- Run `cd yes-no-app-react`
- Run `npm install`
- Run `npm start`
